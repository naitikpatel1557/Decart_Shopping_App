DECÁRT - E-Commerce App 🛒

A beautifully designed, feature-rich e-commerce application built with Flutter and Firebase. DECÁRT offers a seamless shopping experience with a focus on clean UI/UX, real-time database synchronization, and dynamic state management.

🚀 Features
 -> Sleek Splash Screen: A minimalist, timed splash screen ("ALL PRODUCTS. ONE APP.") that transitions smoothly into the main application.
 -> Real-Time Product Feed: Integrates with Firebase Firestore using StreamBuilder to fetch and display products dynamically from the database.
 -> Custom Dynamic App Drawer: A dark-themed, custom-built navigation drawer that adapts to the user's authentication state. It hides/shows sensitive routes (like  -> Address and Orders) and toggles between "Login / Register" and the user's profile information.
 -> Modular Bottom Navigation: A persistent BottomNavigationBar that seamlessly routes between Home, Categories, Wishlist, Orders, and Account screens using state management.
 -> Authentication UI: A professional, Amazon-style login and registration flow featuring secure socket layer (SSL) encryption notices and custom country-code phone inputs.
 -> Responsive AppBars: Dynamic headers that change based on the active tab, including a highly detailed Home header with a search bar, notification badges, and a custom "Support FAQs" button.


🛠️ Tech Stack
 -> Framework: Flutter (Dart)
 -> Backend as a Service (BaaS): Firebase
 -> Database: Cloud Firestore
 -> Architecture: Stateful Widget Management & Stream-based data fetching

📸 UI Highlights
 -> Brand Colors: Deep Teal (#0F4C5C),
 -> Dark Forest Green (#04201E), and vibrant accents.
 -> Typography: Elegant combinations of 'Times New Roman' for branding and modern sans-serif for readability.
 -> Interactive Elements: Features like the "NEW" coupon badge, active/inactive tab icon transitions, and custom text input fields.
